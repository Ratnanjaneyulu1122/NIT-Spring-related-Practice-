	Customization of error messages
	steps to follow:
	-----------------
	1) Create a file errors.properties where define all the custom error messages as key and value pairs
		NotEmpty.employee.name=Employee name should not be empty.
	2)Define the following beans in the spring.xml.
		<bean id="messageSource" class="org.springframework.context.support.ResourceBundleMessageSource">
			<property name="basename" value="errors"/>
		</bean>
		<bean id="validator" class="org.springframework.validation.beanvalidation.LocalValidatorFactoryBean">
			<property name="validationMessageSource" ref="messageSource"/>
		</bean>
